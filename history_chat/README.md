# 聊天归档说明

`history_chat/` 用于保存手动归档的聊天记录。

## 归档规则

- 每次你明确说“归档聊天记录”时，我会在这里新增一个 Markdown 文件。
- 每个归档文件保存一次会话的内容、时间和简要说明。
- 文件名格式：`YYYY-MM-DD_HHMMSS_聊天归档.md`
- 可批量导入当前项目对应的历史 Codex session，并保存为独立 Markdown 文件。

## 当前约定

- 使用 UTF-8 编码
- 默认使用 Markdown 格式
- 存放位置固定为仓库根目录下的 `history_chat/`
- 历史 session 索引保存在 `history_chat/SESSION_INDEX.md`
