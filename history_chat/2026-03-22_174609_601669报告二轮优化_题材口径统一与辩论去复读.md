## 本轮完成的核心工作

### 1. 基于 2026-03-22 新 PDF 复核残留问题

核对文件：

- `reports/601669_分析报告_2026-03-22.pdf`

复核结论：

- 新报告已明显优于 2026-03-21 版本
- “最终建议”“分层建议”“资金证据不完整”“评分解释”等问题已有显著改善
- 但仍存在三类残留问题：
  - 题材标签仍偏生硬，容易把主线相关后排股写成“非主线”
  - 情绪周期与题材轮动对“全市场退潮 vs 局部主线强化”的表述仍容易打架
  - 多空辩论仍有明显复读感，信息增量不足

### 2. 修复题材标签源头逻辑

修改文件：

- [a_share_theme_rotation.py](/home/wing/myproject/TradingAgents-CN/tradingagents/tools/analysis/a_share_theme_rotation.py)
- [theme_rotation_analyst.py](/home/wing/myproject/TradingAgents-CN/tradingagents/agents/analysts/theme_rotation_analyst.py)

修复点：

- 在题材轮动工具源头新增角色映射：
  - `龙头核心 / 前排核心` -> `主线核心`
  - `补涨跟随 / 题材跟风 / 边缘成员` -> 若属于当前主线，优先映射为 `主线外延`
- 输出中保留 `raw_role`，便于区分“工具原始角色”和“给用户看的统一角色”
- 报告正文新增“角色说明”，明确：
  - `主线核心`
  - `主线外延`
  - `非主线`
- 题材分析师提示词进一步约束：
  - 若股属于主线板块但非前排，不得机械写成“非主线”
  - 必须区分“全市场周期”和“局部主线热度”

### 3. 修复“退潮期”与“主线强化/抱团避险”表述冲突

修改文件：

- [a_share_sentiment_analyst.py](/home/wing/myproject/TradingAgents-CN/tradingagents/agents/analysts/a_share_sentiment_analyst.py)

修复点：

- 强制区分：
  - `全市场情绪周期`
  - `局部主线热度`
- 如果弱市中有主线逆势走强，优先写成“局部主线强化或抱团避险”，并明确这不等于全市场全面转强
- 对非龙头个股，必须补一句“板块内轮动/补涨可能”的概率判断，避免完全忽略该情景

### 4. 修复三位辩论分析师输出同质化

修改文件：

- [aggresive_debator.py](/home/wing/myproject/TradingAgents-CN/tradingagents/agents/risk_mgmt/aggresive_debator.py)
- [conservative_debator.py](/home/wing/myproject/TradingAgents-CN/tradingagents/agents/risk_mgmt/conservative_debator.py)
- [neutral_debator.py](/home/wing/myproject/TradingAgents-CN/tradingagents/agents/risk_mgmt/neutral_debator.py)

修复点：

- 三位分析师都新增“去复读”约束
- 每条观点必须尽量使用新的证据点，禁止反复改写“退潮期 / 净流出 / 非龙头”
- 强制分别回应其他立场，而不是各说各话
- 允许在证据不足时直接指出“证据缺口”，而不是硬凑强结论
- 中性分析师必须给出至少一个可执行折中方案，而不是只做概括
